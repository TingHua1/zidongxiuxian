## 入门

### 基础命令

| 命令 | 功能 | 备注 |
| --- | --- | --- |
| `.检测灵根` | 创建角色并生成灵根 / 道号 | 可通过正常发言触发 |
| `.我的灵根` | 查看角色面板 | 显示境界、修为、道号、灵根、宗门等 |
| `.宗门列表` | 查看可加入宗门 | 以实际游戏为准 |

## 规则

### 神魂

- 落败可能触发神魂动荡
- 二次落败可能进入神魂陨落
- 会掉境界、掉修为、掉材料、掉法宝

### 限制

- 每日斗法 / 决斗次数上限
- 对同一目标胜场上限
- 法宝耐久损耗
